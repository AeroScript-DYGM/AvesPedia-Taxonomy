# Bug Fix Prompt

Analyze the BirdTaxonomy project.

Before proposing fixes:

1. Identify root cause.
2. List affected files.
3. Explain why the issue occurs.
4. Show minimal safe fix.

Rules:

- Do not rewrite working code.
- Do not break API contracts.
- Do not change database schema.
- Preserve production compatibility.

Return:

- Root cause
- Risk level
- Proposed fix
- Exact code diff

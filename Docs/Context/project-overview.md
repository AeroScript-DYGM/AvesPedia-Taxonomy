# BirdTaxonomy — Project Overview

## Project Name

BirdTaxonomy

## Description

BirdTaxonomy is a backend-driven software system designed to manage,
query, and maintain scientific bird taxonomy data through REST APIs.

The platform integrates an existing relational database and exposes
taxonomy information using modern software engineering practices.

The system supports:

- Taxonomy rank management
- Scientific taxon registration
- Species metadata management
- CRUD operations through REST endpoints
- Web-based frontend integration

## Main Objective

Provide a scalable and maintainable taxonomy platform that preserves
legacy database contracts while enabling future expansion toward
AI-assisted scientific systems.

## Current Status

Version 1.

Implemented:

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- HTML/CSS/JavaScript frontend
- Swagger documentation
- MonsterASP.NET deployment

## Future Vision

The project is intended to evolve into:

- Multi-project solution
- Cloud-native deployment
- AI-assisted taxonomy workflows
- Data analytics and sustainability research
